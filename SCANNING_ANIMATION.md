## 🎯 动画详解

### 1. 外圈（快速旋转）
```swift
Circle()
    .stroke(LinearGradient(...), lineWidth: 3)
    .rotationEffect(.degrees(-360)) // 反向
    .animation(.linear(duration: 3).repeatForever())
```
**效果**：渐变光圈快速逆时针旋转
**目的**：给人"正在努力工作"的紧张感

### 2. 中圈（脉冲呼吸）
```swift
Circle()
    .fill(RadialGradient(...))
    .scaleEffect(1.0 → 1.08) // 呼吸
    .animation(.easeInOut(duration: 1.2).repeatForever())
```
**效果**：像心跳一样有节奏地跳动
**目的**：1.2秒节奏，模拟心跳感

### 3. 中心图标（慢旋转）
```swift
Image(systemName: "sparkles")
    .rotationEffect(.degrees(360))
    .scaleEffect(1.0 → 1.15)
    .animation(.linear(duration: 4).repeatForever())
```
**效果**：闪烁图标慢速旋转 + 缩放
**目的**：沉稳的主视觉焦点

### 4. 粒子轨道（环绕）
```swift
ForEach(0..<4) { index in
    Circle().fill(accent)
        .offset(y: -50)
        .rotationEffect(.degrees(index * 90 + rotation))
}
```
**效果**：4个点在外围轨道上运行
**目的**：增加运动感和复杂度

### 5. 内波（扫描效果）
```swift
Circle()
    .stroke(accent.opacity(0.3), lineWidth: 2)
    .frame(width: 60 + sin(...) * 10)
    .opacity(0.5 + sin(...) * 0.5)
```
**效果**：中心波纹扩散/收缩
**目的**：模拟雷达扫描效果

## 🎼 节奏设计

### 时间线对比

| 元素 | 周期 | 速度感 | 目的 |
|------|------|--------|------|
| 外圈 | 3s | 快 ⚡ | 紧迫感 |
| 中圈 | 1.2s | 中 💓 | 心跳节奏 |
| 中心 | 4s | 慢 🌟 | 稳定感 |
| 粒子 | 4s | 慢 🔄 | 优雅感 |
| 内波 | 动态 | 变化 📡 | 扫描感 |

### 和谐原则

- **快 + 慢**：外圈快，中心慢 → 层次感
- **大 + 小**：大圈小点 → 主次分明
- **亮 + 暗**：渐变透明度 → 柔和过渡
- **进 + 退**：扩散 + 收缩 → 呼吸感
